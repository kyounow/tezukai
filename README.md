# tezukai — 手取り税金シミュレータ

年収と各種控除から **所得税・住民税・社会保険料** を計算し、**年収 → 手取り** までを可視化する個人向けの税金シミュレータです。あわせて **ふるさと納税** の控除上限額・実績シミュレーション、**住宅ローン控除** などの試算もできます。

🔗 **公開サイト: https://kyounow.github.io/tezukai/**

> ⚠️ **概算ツールです。** 確定申告書類の作成や税理士的な個別助言ではありません。実際の税額・保険料は自治体・年度・個別事情で変わります。
> 🔒 **個人情報は扱いません。** すべてブラウザ内で計算し、入力内容がサーバーに保存・送信されることはありません（バックエンドなし・完全クライアントサイド）。

## 特長

- **2つのモード**: 給与所得者（会社員・パート）／個人事業主（事業所得・国民年金・国民健康保険〔東京特別区の率による概算〕）
- **年収 → 手取り** を内訳バーで可視化（社会保険料・所得税・住民税・手取り）
- **社会保険料**: 協会けんぽ（東京）の標準報酬月額等級表。月給＋賞与を分けた精密計算や、**組合健保（本人負担料率の手入力）**も選択可。子ども・子育て支援金（令和8〜）も反映
- **所得控除**: 基礎・配偶者（特別）・扶養（特定扶養・特定親族特別控除・年少扶養）・社会保険料控除
- **拡張控除**: 医療費控除（通常／セルフメディケーションの有利判定）・生命保険料控除・地震保険料控除・iDeCo（小規模企業共済等掛金）・住宅ローン控除（入居年×新築/中古×性能区分、所得税→住民税の繰越対応）
- **給与以外の所得**: 事業・不動産・雑・配当・一時・総合譲渡の損益通算、所得金額調整控除（給与収入850万円超）
- **ふるさと納税**: 控除上限額の試算（各種所得控除を反映）＋ **実績寄附額から翌年度の住民税を試算**（ワンストップ特例／確定申告、上限超過の警告）
- **年度切替**: 令和7年分（2025）・令和8年分（2026・令和8改正反映）・**令和9年分（2027・暫定）**。既定は確定最新の令和8（令和9は暫定バナー付き）
- ライト／ダークモード対応・日本語UI

## 計算スコープ

| 区分 | 内容 |
|---|---|
| コア | 給与所得控除／事業所得（青色申告特別控除）→ 所得控除 → 課税所得 → 所得税（復興特別所得税2.1%込み）→ 住民税（所得割＋均等割＋森林環境税＋調整控除）→ 社会保険料 → **手取り** |
| 拡張 | ふるさと納税の控除上限・実績シミュレーション、住宅ローン控除、医療費控除、iDeCo、生命保険料控除、地震保険料控除 |

## 開発

```bash
npm install        # 依存インストール
npm run dev        # ローカル開発サーバー（Vite）
npm run test       # Vitest（計算ロジックのテスト）
npm run test:watch # Vitest（watch モード）
npm run typecheck  # 型チェックのみ（tsc -b --noEmit）
npm run build      # 本番ビルド（tsc -b → vite build → dist/）
npm run preview    # 本番ビルドのローカルプレビュー
```

`main` への push で GitHub Actions（[`.github/workflows/deploy.yml`](.github/workflows/deploy.yml)）がビルド → テスト → GitHub Pages へ自動デプロイします。本番ビルドのベースパスは [`vite.config.ts`](vite.config.ts) で `/tezukai/` に設定。

## アーキテクチャ

- **`core/`** … UI 非依存の計算ドメイン層（純粋関数）。所得区分・各控除・所得税/住民税/社会保険料・手取り、ふるさと納税、住宅ローン控除など。`TaxTable` を注入されて計算し、Vitest で検証する中心。
- **`data/taxTables/`** … 年度別の税率・控除の定数テーブル（出典コメント付き）。`types.ts`（集約型 `TaxTable`）＋ `<年度>.ts` ＋ `index.ts`（レジストリ）。新年度の追加手順は [`data/taxTables/README.md`](data/taxTables/README.md) を参照。
- **`app/`** … React の UI（入力フォーム・結果表示・年度セレクタ）。`core/` を呼ぶだけで、計算は UI に書かない。

税率・控除額は **年度別データとして分離** し、法改正時は新しい `<年度>.ts` を足すだけで対応できる設計です。計算ロジックは TDD（テスト駆動）で、公式資料に基づくケースを先に書いてから実装しています。

## 技術スタック

React 19 ＋ TypeScript ＋ Vite 6 ＋ Vitest 3。バックエンドなしの静的サイトとして GitHub Pages にデプロイ。

## 出典

税率・控除額は国税庁・総務省・各自治体・協会けんぽ・日本年金機構等の一次情報に基づきます。年度別の根拠は [`data/taxTables/sources-2025.md`](data/taxTables/sources-2025.md) / [`sources-2026.md`](data/taxTables/sources-2026.md) を参照してください。
