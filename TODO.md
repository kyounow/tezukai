## 社会保険料（近似）

- [x] **賞与分離モード**: 月給＋賞与を分けて標準報酬月額・標準賞与額（上限 健保573万/年・厚年150万/月×回数）で社保を精密計算。簡易モード（報酬月額＝年収/12）も選択可。随時改定/定時決定・賞与支給月の精密配分は未反映。
- [x] **組合健保（手入力）**: 給与所得者モードで「組合健保」を選ぶと本人負担の健保・介護・子ども子育て支援金（令和8〜）料率を直接入力して試算可（折半せず本人負担率をそのまま適用）。厚年・雇用は不変。**既定はNTT健康保険組合（令和8年度・本人負担：健保4.56%・介護0.77%・支援金0.115%）**。
- [x] **子ども・子育て支援金（令和8年度〜）**: 医療保険料に上乗せ・本人折半。協会けんぽは令和8で0.23%（労使合計）を健保に加算済み、組合健保は本人負担率を手入力。※**子ども・子育て拠出金**（事業主全額0.36%・厚年併徴・本人負担なし）は別物で対象外。
- ~~**協会けんぽの支部差**~~（**対応不要**と判断・2026-06-28）: 東京固定。支部差は小さく（令和7：最低沖縄9.44%〜最高佐賀10.78%）、料率差の大きい組合健保は手入力で対応済みのため。介護料率は全国一律。
- [x] **65歳以上の社保**: 厚年は70歳到達で0、健保は75歳で後期高齢者医療へ移行し0、介護は40〜64歳のみ（65歳以上は第1号で給与天引きでなく別途）、雇用は継続。75歳以上の後期高齢者医療保険料は別途（未対応・注記）。出典: 日本年金機構・厚労省。
- [ ] **到達年の月割り**: 70/75歳に到達する年も月割りせず年額（月額×12）で概算（到達月前後の計上月数の変化・厚年の誕生日前日基準・1日生まれ特例は未対応）。

## 税額の近似・未対応

- [x] **住民税の自治体差**: 級地区分（1/2/3級地）で非課税限度額を切替（級地率1.0/0.9/0.8を基本額35万・加算21万/32万に乗算、+10万は国一律）、均等割の超過課税を手入力で上書き可（森林環境税1,000円は固定）。出典: 総務省・自治体・税研。
- [ ] **調整控除の人的控除差**: 主要控除のみ・本人高所得ティアや配偶者特別控除の所得帯差は基準値で近似。令和7改正後の差額は要再確認。
- [x] **給与以外の所得・損益通算**: 事業・不動産・雑・配当・一時・総合譲渡（総合課税）を所得区分ごとに入力し損益通算に対応。**分離課税（上場株式等・土地建物の譲渡等）は対象外**。所得金額調整控除（給与収入850万超の子育て・特別障害者等）も対応済み。
- [x] **拡張控除**: 医療費・生命保険料・**地震保険料**・iDeCo・住宅ローン控除を実装済み。ふるさと納税上限にも反映（住宅ローン控除は税額控除なので上限の基礎＝住民税所得割は不変、所得控除は上限が下がる）。
- [x] **ふるさと納税**: 控除上限額に加え、実績寄附額からの**翌年度住民税の試算**（ワンストップ特例／確定申告、上限超過の警告）に対応。ワンストップの申告特例控除は概算（特例控除額×割合の一括計算。市民税分・県民税分の按分・各別切上げはしない）。
- [ ] **住宅ローン控除の範囲**: 現行制度（令和4〜7入居・0.7%）のみ。**令和3以前の1%制度は対象外**。新築「その他（省エネ基準なし）」の令和6・7入居経過措置（借入2,000万・10年）は未対応（0扱い）。**控除期間（新築13年・中古10年）の経過判定に対応**（入居年から期間を過ぎた年・入居前は0。国税庁 No.1211-1/1211-3）。※現行UIは入居2022〜・税年度2025〜のため期間切れは将来年度で発火。
- [ ] **生命保険料控除**: 子育て世帯の一般生命保険(新)の拡充（4万→6万・所得税）は**令和8年分のみ反映**（令和7年分は対象外）。医療費控除の「総所得金額等」は給与所得＝総所得として近似。iDeCo は掛金額をそのまま受付（加入区分別の拠出限度バリデーション未実装）。
