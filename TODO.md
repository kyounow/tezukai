## 社会保険料（近似）

- [x] **賞与分離モード**: 月給＋賞与を分けて標準報酬月額・標準賞与額（上限 健保573万/年・厚年150万/月×回数）で社保を精密計算。簡易モード（報酬月額＝年収/12）も選択可。随時改定/定時決定・賞与支給月の精密配分は未反映。
- [x] **組合健保（手入力）**: 給与所得者モードで「組合健保」を選ぶと本人負担の健保・介護・子ども子育て支援金（令和8〜）料率を直接入力して試算可（折半せず本人負担率をそのまま適用）。厚年・雇用は不変。**既定はNTT健康保険組合（令和8年度・本人負担：健保4.56%・介護0.77%・支援金0.115%）**。
- [x] **子ども・子育て支援金（令和8年度〜）**: 医療保険料に上乗せ・本人折半。協会けんぽは令和8で0.23%（労使合計）を健保に加算済み、組合健保は本人負担率を手入力。※**子ども・子育て拠出金**（事業主全額0.36%・厚年併徴・本人負担なし）は別物で対象外。
- ~~**協会けんぽの支部差**~~（**対応不要**と判断・2026-06-28）: 東京固定。支部差は小さく（令和7：最低沖縄9.44%〜最高佐賀10.78%）、料率差の大きい組合健保は手入力で対応済みのため。介護料率は全国一律。
- [x] **65歳以上の社保**: 厚年は70歳到達で0、健保は75歳で後期高齢者医療へ移行し0、介護は40〜64歳のみ（65歳以上は第1号で給与天引きでなく別途）、雇用は継続。75歳以上の後期高齢者医療保険料は別途（未対応・注記）。出典: 日本年金機構・厚労省。
- [ ] **到達年の月割り**: 70/75歳に到達する年も月割りせず年額（月額×12）で概算（到達月前後の計上月数の変化・厚年の誕生日前日基準・1日生まれ特例は未対応）。

## 税額の近似・未対応

- [x] **住民税の自治体差**: 級地区分（1/2/3級地）で非課税限度額を切替（級地率1.0/0.9/0.8を基本額35万・加算21万/32万に乗算、+10万は国一律）、均等割の超過課税を手入力で上書き可（森林環境税1,000円は固定）。出典: 総務省・自治体・税研。
- [ ] **調整控除の人的控除差**: 基礎・配偶者・扶養に加え、障害者（普通1万/特別10万/同居特別22万）・ひとり親5万・寡婦1万・勤労学生1万の差を反映済み。**ひとり親（父）は法定1万だが父母を区別せず5万で近似**。本人高所得ティアや配偶者特別控除の所得帯差は基準値で近似。令和7改正後の差額は要再確認。
- [x] **給与以外の所得・損益通算**: 事業・不動産・雑・配当・一時・総合譲渡（総合課税）を所得区分ごとに入力し損益通算に対応。**分離課税（上場株式等・土地建物の譲渡等）は対象外**。所得金額調整控除（給与収入850万超の子育て・特別障害者等）も対応済み。
- [x] **拡張控除**: 医療費・生命保険料・**地震保険料**・iDeCo・住宅ローン控除を実装済み。ふるさと納税上限にも反映（住宅ローン控除は税額控除なので上限の基礎＝住民税所得割は不変、所得控除は上限が下がる）。
- [x] **ふるさと納税**: 控除上限額に加え、実績寄附額からの**翌年度住民税の試算**（ワンストップ特例／確定申告、上限超過の警告）に対応。ワンストップの申告特例控除は概算（特例控除額×割合の一括計算。市民税分・県民税分の按分・各別切上げはしない）。
- [ ] **住宅ローン控除の範囲**: 現行制度（令和4〜8入居・0.7%）のみ。**令和3以前の1%制度は対象外**。新築「その他（省エネ基準なし）」の令和6・7入居経過措置（借入2,000万・10年）、および令和8入居の新築「その他」の経過措置（借入2,000万・10年）は未対応（0扱い）。**控除期間の経過判定に対応**（新築13年・中古は令和7以前入居10年／**令和8以降入居は省エネ性能あり〔認定・ZEH・省エネ基準適合〕で13年・その他10年**。入居年から期間を過ぎた年・入居前は0。国税庁 No.1211-1/1211-3、財務省 令和8年度税制改正の大綱）。※現行UIは入居2022〜・税年度2025〜のため中古13年化・期間切れは将来年度で発火。
- [ ] **生命保険料控除**: 子育て世帯の一般生命保険(新)の拡充（4万→6万・所得税）は**令和8・9年分に反映**（令和7年分は対象外の時限特例。令和8年度改正で令和8年分のみ→令和9年分まで1年延長。出典: 財務省 令和8年度税制改正の大綱）。医療費控除の「総所得金額等」は給与所得＝総所得として近似。iDeCo は掛金額をそのまま受付（加入区分別の拠出限度バリデーション未実装）。

## 育児休業（近似）

- [x] **育休手取り**: 給与所得者モードで育休（開始日・終了日）を入力すると、課税給与を育休日数ぶん減額し、育児休業給付金（180日まで67%・181日〜50%）＋出生後休業支援給付金（両親14日以上で13%・最大28日）を**非課税**で手取りに加算、健保・厚年・介護を社保免除月数（月末育休 or 同月内に開始・終了する育休が14日以上）ぶん免除して概算。出典: 厚労省・ハローワーク・日本年金機構。
- [x] **分割育休**: 期間を複数登録可能（「育休期間を追加」）。給付率67%/50%は全期間を**通算**した育休日数で判定、社保免除は全期間の**和集合**で月判定（同一月の分割は合算14日で免除、月をまたぐ育休の復職月は14日ルール対象外、期間の重なりは二重計上しない）。※制度上の分割回数の上限（育休 原則2回・産後パパ育休2回）はUIで強制せず、入力した期間どおり概算する。
- [x] **年またぎの育休**: 税は暦年課税のため、選択中の税年度の暦日のみを当年分として計算（給与減・社保免除・給付金を当年内にクリップ）。給付率67%/50%は育休開始からの通算で判定するので、前年までの育休日数を踏まえて当年分の率を決める。育休が年をまたぐ場合はUIに注記を出し、年度を切り替えて各年を確認する運用（年単位の概算）。
- [ ] **育休の近似**: 賃金日額＝月給/30（実際は育休前6か月賃金/180）、給与減は日割り（月給×当年N/30）近似。給付金は支給単位期間（30日）ごとの端数処理をせず合計floor。賞与の社保免除はチェックで簡易対応。
- [ ] **賃金日額上限16,110円は令和7.8〜令和8.7の値**（毎年8月改定）。令和8/9は令和7を流用（`childcareLeaveBenefit` を年度データで差し替え可）。養育期間の年金みなし措置は手取り無関係で対象外。

## 実額（源泉徴収票）モード

見込み試算に加え、給与所得者は源泉徴収票の実額（支払金額・社会保険料等の金額・iDeCo・源泉徴収税額）から、
確定申告した場合の所得税の年税額（試算）と源泉徴収税額の差＝**還付/追徴の目安**、翌年度住民税の見込み・手取り・
ふるさと納税上限（実績ベース）を計算する。給与所得は別表第五（電算機特例）で源泉徴収票の「給与所得控除後の金額」に
一致させ、社会保険料は実額オーバーライドで料率計算をスキップする。**「確定」の語は使わない**（所得税は申告ベースの試算、
住民税は自治体決定額と端数処理等で差が出る計算値のため）。

- [ ] **源泉徴収票の画像読み取り（OCR）**: 券面の自動読み取りは未対応（手入力のみ）。
- [ ] **複数の給与（乙欄）・年途中の入退社の月割り**: 単一の年末調整済み源泉徴収票を前提。複数支払者（主・従）や乙欄、年途中の入退社の月割りは未対応。
- [ ] **個人事業主の報酬源泉徴収（10.21%）の精算**: 報酬から源泉徴収された所得税の精算は v1 対象外（実額タブの源泉徴収税額欄は給与所得者のみ表示）。
- [ ] **検算欄**: 源泉徴収票の「給与所得控除後の金額」「所得控除の額の合計額」を券面値と突き合わせる検算欄は未実装。
- [ ] **源泉徴収票の欄位置ミニ図**: どの欄の数値をどこに入力するかの視覚ガイド（ミニ図）は未実装（ヒント文で代替）。
